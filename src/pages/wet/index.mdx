## mdxファイル上での画像の最適化

### 画像の読み込み
```js
import test from '../../assets/test.JPG';
import { Image } from 'astro:assets';
```
まず、mdxファイルの冒頭で画像を読み込むためのコードを記述します。  

### 画像の表示
```js
<Image src={test} alt="test" format="avif" width={1080} />
```
こんな感じのタグを張ると画像が最適化されます。

### 画像の圧縮率
<Image src={test} alt="test" format="avif" width={1080} />
例えばこの画像の元のサイズは6.98 MBですが、avif形式で1080pxに圧縮すると22.6KBになります。
